Спецификация визуального компонента "Список"

  • Введение

    • Назначение
      • Компонент применяется для наглядного отображения и редактирования связанных списков, осуществляет вывод на экран данных в виде списков, с возможностью сортировки. Список используется для упрощения анализа и поиска элементов, а так же для редактирования необходимых значений.
    • Область действия
      • Элемент предназначен для работы через браузер ПК.
        См. поддерживаемые версии браузеров в Приложении 1.
    • Краткий обзор

  • Общее описание

    • Взаимодействие продукта (с другими продуктами и компонентами)
      • Загрузка списков в определенном формате (см. Приложение 2) происходит после зарпроса на сервер.
      • Редактирование элементов списка происходит при помощи взаимодействия с сервером. После подтверждения отправки формы, содержащей новое значение, формируется запрос к серверу на изменние данных, далее пользователь получает оповещение об успешном/не успешном изменении и происходит загрузка нового представления списка.
      • Режим поиска позволяет ввести требуемое значение, далее после отправки формы, элемент выводит на главый экран результаты поиска с виде нумерованого списка.
    • Функции продукта (краткое описание)
      • Компонент "Список" так же поддерживает наличие внутрених связей между разными списками, возможность загрузки данных из разных списков в список, открытый в текущий момент. Поиск элементов осуществляется по уникальному индентификатору (id), присваемому каждому из элементов, при создании.
      • Предусмотрена возможность изменения значения ячеек, как на экране отображения списка, так и на экране вывода результатов поиска по знаяению.
      • Компонент имеет возможность поиска по значению ячейки и возможность сортировки резльтатов поиска.
      Характеристики пользователя
      Ограничения

  • Описание внешних интерфейсов

    • Интерфейсы пользователя
      • Список поддерживает настройку графического интерфейса пользователя через панели управления в правой, левой и нижней частях экрана.
      • Возможность скрывать и отображать столбцы списков реализована в нижней части экрана
      • Настройки сортировок и параметров отображения базовых списков расположена в левой чати экрана
      • Настройки сортировок и параметров отображения основных списков расположена в правой чати экрана
      • На заголовке списка, на каждом из столбцов есть кнопка изменения сортировки: "По возрастанию", "По убыванию"
    • Интерфейсы взаимодействия

  • Примеры реализации и использования

    • Список задач и вложенный список назначенных сотрудников
      • На правой панели управления осуществляется форматирование списка задач, который содержит список id назначенных сотрудников, колонка "Назначен" заполняется элементами из списка "Сотрудники" в соответсвии с идентификатором. Список поддерживает сортировку по всем колонкам (по возрастанию, по убыванию) по датам, бюджету, названию проекта, по id.
